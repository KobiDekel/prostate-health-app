# 🥗 Prostate Health & Longevity AI App

Personalized nutrition assistant for Gleason 3+3/3+4 prostate cancer. Grounded in research to inhibit cancer while protecting Heart, Liver, Kidney, and Brain.

## 🛠️ Features
- **Daily AI Menus:** Grounded in research sources.
- **Dual Reasoning:** Explains Cancer inhibition + Systemic benefits.
- **Smart Notifications:** Real-time health tips.

## 🚀 Built with
- HTML/JS (Frontend)
- Node.js (Vercel API)
- Google Gemini AI (Engine)
